# ✈️ Fare Calculation Web Tool

A simple and professional web-based tool designed for GDS fare calculation practice and ticketing workflow support.

## 📌 Project Description

This website is built to assist in airline fare calculation tasks.  
It provides a clean interface with aviation-themed design for practical use in ticketing and GDS operations.

## 🚀 Features

- Simple and clean user interface
- Aviation themed background
- Fast loading static webpage
- Easy to modify and extend

## 🛠️ Technologies Used

- HTML5
- CSS3
- javaScript


## 💻 How to Use

1. Download or clone the repository  
2. Open `index.html` in your browser
